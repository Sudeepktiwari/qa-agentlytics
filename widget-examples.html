This will work with just the script src - all other attributes are optional: 

<script src="http://localhost:3001/api/widget"></script>

üé® Full Configuration Options

Here are all available customization options with their default values:

üìù Basic Configuration

  • data-api-key: API key for authentication
    Default: Built-in fallback key
  • data-theme: Color theme
    Options: blue, green, purple, orange, dark, custom
    Default: "blue"
  • data-size: Widget size
    Options: small, medium, large
    Default: "medium"
  • data-position: Widget position
    Options: bottom-right, bottom-left, top-right, top-left
    Default: "bottom-right"

üí¨ Chat Customization

  • data-chat-title: Title shown in chat header
    Default: "Chat with us"
  • data-welcome-message: Initial welcome message
    Default: "Hello! How can I help you today?"
  • data-button-text: Text/emoji on the chat button
    Default: "üí¨"
  • data-custom-color: Custom hex color (when theme="custom")
    Default: "#0070f3"

üé§ Voice Settings

  • data-voice-enabled: Enable text-to-speech
    Options: true, false
    Default: true
  • data-voice-gender: Voice gender
    Options: female, male
    Default: "female"
  • data-auto-open-proactive: Auto-open chat for proactive messages
    Options: true, false
    Default: true

üöÄ Example Usage

Example 1: Minimal (uses all defaults)

<script src="http://localhost:3001/api/widget"></script>

Example 2: Basic customization

<script 
    src="http://localhost:3001/api/widget"
    data-theme="green"
    data-size="large"
    data-chat-title="Help & Support">
</script>

Example 3: Full customization

<script 
    src="http://localhost:3001/api/widget"
    data-api-key="your-api-key-here"
    data-theme="custom"
    data-custom-color="#ff6b6b"
    data-size="large"
    data-position="bottom-left"
    data-chat-title="Customer Support"
    data-welcome-message="Hi! Need help? I'm here to assist you!"
    data-button-text="üÜò"
    data-voice-enabled="true"
    data-voice-gender="male"
    data-auto-open-proactive="false">
</script>

Example 4: Voice disabled

<script 
    src="http://localhost:3001/api/widget"
    data-theme="dark"
    data-voice-enabled="false">
</script>

üéØ Testing Proactive Messages

Once the widget loads, you can test proactive messages in the browser console: 

// Send a proactive message
window.appointyChatbot.sendProactiveMessage('Hello! This is a test message!');

// Check if chat is open
console.log('Chat open:', window.appointyChatbot.isOpen());

// Open/close chat programmatically
window.appointyChatbot.openChat();
window.appointyChatbot.closeChat();
